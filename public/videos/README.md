# Courage the Cowardly Dog - Video Storage

## How to Add Videos

Add all 71 episode video files to this folder with the exact naming format:

\`\`\`
Courage- Chú Chó Nhút Nhát (Thuyết Minh)(1).mp4
Courage- Chú Chó Nhút Nhát (Thuyết Minh)(2).mp4
Courage- Chú Chó Nhút Nhát (Thuyết Minh)(3).mp4
...
Courage- Chú Chó Nhút Nhát (Thuyết Minh)(71).mp4
\`\`\`

## File Structure

\`\`\`
public/
└── videos/
    ├── Courage- Chú Chó Nhút Nhát (Thuyết Minh)(1).mp4
    ├── Courage- Chú Chó Nhút Nhát (Thuyết Minh)(2).mp4
    ├── Courage- Chú Chó Nhút Nhát (Thuyết Minh)(3).mp4
    ├── ...
    └── Courage- Chú Chó Nhút Nhát (Thuyết Minh)(71).mp4
\`\`\`

## Important Notes

- The filename format must match exactly, including parentheses and spaces
- The number in parentheses corresponds to the episode number (1-71)
- Video files are automatically loaded based on the episode number
- After uploading videos, the website will immediately start playing them

## Supported Formats

- MP4 (.mp4)
- WebM (.webm)
- OGG (.ogg)
